# 所見即所抓
## 使用方法
將貼有april tag的木塊放在可視區, 執行<br>
python cube_m3.py<br>
jetARM就會找有april tag的木塊,抓取,右轉90度,放下木塊,歸零.<br>
## 參數 pixels_per_30mm怎麼來的 ?
將jetARM歸零,放上方格板,將木塊放在30mm的位置,執行<br>
python cube_m3.py wait<br>
因為高度的關係,april tag的座標會向外偏.<br>
